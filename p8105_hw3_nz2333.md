\#Question3

``` r
instacart %>%
  filter(aisle %in% c("baking ingredients", "dog food care", "packaged vegetables fruits")
  )%>%
  group_by(aisle, product_name)%>%
  summarize(
    item_num = n()
  )%>%
  mutate(
    item_rank = min_rank(-item_num)
  )%>%
  filter(item_rank < 4) %>%
  select(-item_rank) %>%
  knitr::kable()
```
